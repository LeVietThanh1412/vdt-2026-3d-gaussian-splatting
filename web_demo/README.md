# Web Demo

This folder hosts a minimal viewer for the exported glb.

Steps:
1) Copy your output .glb into web_demo/model.glb
2) Run: python3 web_demo/serve.py
3) Open http://localhost:8000 in a browser
4) On mobile, open the same URL via your LAN IP for AR

If you use a different file name, update the src attribute in index.html.
